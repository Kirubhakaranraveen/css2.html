What is Css?

  1. Css stands for Cascading Style Sheets.

  2. Css describe how HTML elements are to displayed on screen,paper,or in other media.

  3. Css is not necessaryCss saves a lot of work.It can control the layout of multiple Web pages all at once

  4. External Stylesheets are stroed in Css files.

Css can be added to HTML documents in 3 ways:

  • Inline- by using the style attribute inside HTML elements1
  • Internal-by using a style element in the head section
  • External-by using a link elememt to link to an external css files
  • The most common way to add Css. is to keep the style in external Css files.However in this tutorial we will use inline and internal styles,bcause this is casier to demonstrate and easier for you to try it yourself.

when comparing Css class vs ID,the different is that Css class applies a style to multiple elements.ID,on the other hand,applies a style to one unique element.ID is also specializ in that you can use a special URL to link directly to an element and it's used by JavaScript.

In Css, selectors are used to target a special or range of element on a Web page.once an element has been target,a style or set of style can be applied to the element.

There is wide range of selectors available.Two of the most commonly used are class and ID.Both are used to target element to which a style should be applied.
